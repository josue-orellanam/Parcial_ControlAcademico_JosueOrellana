## 1. Descripción del Proyecto
Este repositorio contiene el desarrollo del sistema de Control Académico, integrando el control de versiones con Git, la estructura de una base de datos relacional y consultas SQL para la extracción de información académica. [cite: 17]

## 2. Estructura del Proyecto
- **/database**: Carpeta que contiene el archivo `script.sql` con el diseño de tablas, inserciones y consultas. [cite: 25, 26]

## 3. Modelo de Base de Datos
La base de datos se denomina `control_academico` y consta de tres entidades principales: [cite: 54]
- **Alumno**: Información personal y estado del estudiante. [cite: 32]
- **Curso**: Catálogo de asignaturas y créditos. [cite: 39]
- **Inscripcion**: Relación de muchos a muchos entre Alumnos y Cursos. [cite: 44]
